# Проект 1. Приложение для анализа банковских операций

## Описание
Проект "курсовая работа " - написание функции на языке Python

1. Модуль reports.py - Содержит функцию  которая возвращает траты по заданной категории за последние три месяца (от переданной даты).
2. Модуль services.py - Содержит функцию в которой траты будут округляться, и разница между фактической
    суммой трат по карте и суммой округления будет попадать на счет Инвесткопилки.
3. Модуль views.py - Содержит много доп функции которые помогают основной функции . Выдает 
Приветствие в формате 
"???"
, где 
???
 — «Доброе утро» / «Добрый день» / «Добрый вечер» / «Доброй ночи» в зависимости от текущего времени.
По каждой карте:
последние 4 цифры карты;
общая сумма расходов;
кешбэк (1 рубль на каждые 100 рублей).
Топ-5 транзакций по сумме платежа.
Курс валют.
Стоимость акций из S&P500.
4. Так же есть  основной файл в папке data с информации по всем транзакциями
